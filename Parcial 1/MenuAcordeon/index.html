account_balance_wallet Modelo de Caja HTML

Todos los elementos de los documentos HTML (páginas) se representan mediante cajas rectangulares. Cada vez que se inserta una etiqueta HTML, se crea una nueva caja rectangular que encierra el contenido de la etiqueta.

Las partes del modelo de Caja son:
Content: El contenido del cuadro, donde aparecen el texto y las imágenes.
Padding: El área alrededor del contenido. es decir entre el contenido y el borde.
Border: El borde envuleve el padding y el content.
Margin: El area transparente afuera del borde y que las separa de la demas cajas.

reorder Flujo Normal del Documento

Es la manera en la que los elementos son desplegados en el documento. Dependiendo del tipo de elemento, estos se colocarán de la siguiente manera:
Elementos de Bloque. Siempre empiezan en una nueva línea y no se sitúan uno al lado del otro, también, cada línea se completa con un único elemento, aún si su contenido es mínimo, además, el flujo es de arriba hacia abajo.
Elementos de Línea. Se colocan en línea hasta que sus anchos lo permitan y cuando completen una línea saltan a la siguiente. Aquí el flujo es de izquierda a derecha.

info Elementos en flujo y fuera de flujo

En Flujo: Los elementos en flujo reciben su espacio y ese espacio es respetado por los otros elementos en el flujo.
Fuera de flujo: Si un elemento es flotado o posicionado absoluto entonces este elemento estará fuera de flujo y su espacio no serárespetado por los otros elementos que si están en el flujo.

sticky_note_2 Tipos de Selectores CSS

Básicos. Selector universal, de etiqueta, de identifiador y de clase.
Relacionales. Selecciona elementos basándonse en su relacion con otros elementos.
Pseudo Selectores. Son palabras clave agregadas a un selector para hacerlo más específico.
Atributos. Selecciona elementos basándose en sus atributos y sus valores.

subject Cascada

La cascada se refiere a que las declaraciones de estilo pueden estar en varios sitios y dependiendo este sitio, afectará a mas o menos elementos.
Cuando dos o mas reglas se aplican a un mismo elemento: El navegador las aplica todas y prevalecerá siempre la última regla escrita.
Cuando dos o mas reglas entran en conflicto: Existen reglas para definir la precedencia.

css CSS Media Queries

Las Media Queries son parte fundamental del diseño web, ya que son la base del diseño responsivo o adaptable.Mediante el uso de estas podemos hacer que un mismo marcado html pueda adpatarse a diferentes dispositivos y tamaños de pantalla.
El objetivo de usarlos en una hoja de estilo CSS es que se ejecute una regla o un conjunto de reglas solo si se cumple una condición determinada, como por ejemplo el ancho de pantalla.
